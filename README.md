**Setup**
<br>

- .env requirements
- PORT - 3003
- Running the app
- npm start
- Endpoint:

1- food

```
let obj = {
type: 'banana',
color: 'yelow',
};

    http://localhost:3003/api/v1/food
```

2- clothes

```

let obj = {
type: 'T-shirt',
color: 'white',
};
http://localhost:3003/api/v1/clothes

```

- Returns : json with requested data

```

{
"domain": "https://ahmadosbeh-basic-express-serve.herokuapp.com/",
"status": "running",
"port": 3003
}

```
